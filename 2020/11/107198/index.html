SWISS-KNIFE瑞士刀插件

发表于 2020-11-10 更新于 2020-11-17 分类于 AE 阅读次数： Valine：
本文字数： 2.9k 阅读时长 ≈ 3 分钟

…

插件下载

分组

从上至下分别为

设置(Options)、视差(Parallax)、交互(Interactions)、占位符(Placeholders)、辅助(supports)

Options - 设置

  • HelpTips On/Off-鼠标悬浮在按钮上会显示帮助信息
  • Icons On/Off-Icon是否需要显示
  • Column On/Off-行列排列模式
  • Options-可以选择主菜单在界面的哪个方向

Parallax - 视差

Parallax3D-视差3D

将选中的图层保持摄像机默认起点的视差大小，按一定方法分配到设定的深度空间上

  • 直接点击Parallax3D，按随机的方式分配在设定的深度中的位置
  • Alt+点击Parallax3D，等距离分配
  • Shift+点击Parallax3D，对于部分图层所处位置不满意时，调整速度空间，然后重新分配选中的部分不满意的图层的速度位置

Parallax2D-视差2D

以2D的方式模仿3D视差移动效果

  • 移动控制层即可看见效果
    • Random Seed-随机种子
    • Randomness-随机幅度
    • Index Parallax-切换基于索引的空间分配(勾选)，或者基于随机的空间分配
    • Range(Random Parallax)-random模式中各个图层之间的速度差异
    • Index Offset(Index Parallax)-Index模式中处于0,0,0位置的图层的索引
    • Center-移动整个视差图层
  • 每个受控图层可以设置自身的偏移量(模拟Z深度)

Extuder-挤压

将选中的图层挤压出纯色厚度

可设置参数：

  • Extursion color-纯色的颜色
  • Extursion opacity-纯色的透明度
  • Extursion Depth-挤压厚度
  • Group-单独设置每个面的颜色
    • Unicue Colors-开关单独设置颜色功能
    • x side-各个方向的颜色设置

STACK-堆叠

类似于形状层的中继器，具有更灵活的选项

  • 直接点击STACK-2d堆叠

  • Alt+点击STACK-3d堆叠

  • Isometric-基于等测轴的堆叠

  • Carusel-跑马灯(自带动画以及推荐C4D模式)

可控属性

Interactions - 交互

INTERACTOR-互动

模拟类似鼠标与UI按钮交互

使用方法：

注意图层中心点位置，这是触发动画的关键参考

  • 制作相关的动画，安照“未触发-触发”时间轴制作，速度无关，速度曲线有关
  • 选中所有需要设置INTERACTOR的关键帧
  • 加选一个“鼠标”，鼠标的位置是基于鼠标图层的中心而非图层可见部分
  • 点击INTERACTOR（触发盒子大小统一设置）
    Alt+点击INTERACTOR（每个图层的触发盒子大小单独设置）

DEPENDENCER-依靠

模拟进度条与动画，即可以使用进度条控制动画的进程

使用方法：

注意scope的选择

单图层无时间偏移动画-Propertiy

单独层时间偏移动画-Layer

多图层动画-Composite

  • 选择“时间轴”图层，确保时间轴图层的动画基于"起始位置-终点位置"做好动画
  • 点击DENPENDENCER,之后此按钮会边红并显示APPLY
  • 选择Scope的模式
  • 选择所有需要设置的关键帧
  • 点击APPLY
  • 可以取消“时间轴”的关键帧，新生成的空对象图层有一个效果器，效果器上有入点和出点的设置，即“时间轴”的开始与结束，checkbox勾选之后表示起点与距离

ANIMARKER-动画标记

通过marker标记控制动画的起始

使用方法：

这个操作的最小单位是图层，而非关键帧

图层标记通过小键盘*添加

  • 选中所有想要设置动画标记的关键帧
  • 点击ANIMARKER,新生成的空对象图层的轴上有一个动画标记，即最早关键帧出现的时间
    Alt+点击ANIMARKER,所有被选中的图层都分别添加了一个标记，即图层自己的最早关键帧出现时间
  • 通过增加标记增加动画播放的次数和动画播放的起始时间

可以通过对reverse做关键帧达到动画正向播放-反向播放的目的

OFFSET-偏移

适用于大量阵列的偏移动画

圆心模式下，偏移是向外扩散的

使用方法：

  • 选定所有需要做偏移的图层(无需关键帧)
  • 点击OFFSET
  • 对新生成的空对象做基础属性的动画会产生偏移

对于偏移控制的空对象，有如下可控参数

  • Time offset-偏移帧数
  • Inherit Value from Main Layer
  • Inherit Position from Main Layer-勾选开启网格模式
  • step-有多少个连续图层具有相同的偏移

  • Grid Options-网格模式设置(2维阵列)

    可以通过对阵列元素复制产生新的阵列元素

    • X/Y Offset-逐个元素的偏移距离
    • Columns/Rows Order-优先排满行/列上的元素
    • Columns/Rows Count-行/列上的元素最大个数
    • Grid/Radia Grid-控制器在首行首列，偏移按照顺序偏移/在中心，偏移按照放射状进行
    • 1/4 in center-控制器在四个元素的中间

REVOLVER-旋转器

只对图层旋转属性有影响

与INTTERACTOR类似，使用一个“鼠标”，以图层中心点之间的距离作为依据，对受控图层的旋转属性进行控制

使用方法:

  • 选中所有需要设置REVOLVER的图层
  • 加选一个“鼠标”，鼠标的位置是基于鼠标图层的中心而非图层可见部分
  • 点击REVOLVER（触发盒子大小统一设置）
    Alt+点击REVOLVER（每个图层的触发盒子大小单独设置）

Placeholder - 占位符

简化了使用预合成、Alpha模板和素材的流程

PLACEHOLDER

分离形状

分离同一个形状图层下的多个形状

批量遮罩

先选择遮罩，然后加选被遮罩的图层，点击PLACEHOLDER即可

注意遮罩数量要和被遮罩数量一致

可以通过设置的合成上的效果器对图层的位置进行操作

批量预合成

选择需要预合成的图层，Alt+PLACEHOLDER即可

PINER-贴

将图层贴到透视空间上

  • 划出透视空间的矩形形状层
  • 选中形状层点击PINER，此时会变红
  • 选中需要放置到空间中的图层，点击红色的APPLY

可以通过参数进行调整

SWITCHER-切换器

对一系列图层进行选择

比如口型：

  • 选中所有口型，调整好中心点和大小
  • 点击SWITCHER
  • 新生成的合成上的选择器进行口型选择，选择器的数字对应预合成中的Index

ALPHA MATTE-Alpha遮罩

  • 把位于所选图层Index最小的图层克隆为其余图层的AlphaMatte

  • 使用Alt+ALPHA MATTE会将最小的图层克隆并位移到其余图层的中心处并设置AlphaMatte

SLIDESHOW-幻灯片

将选择的一系列图层以幻灯片的形式切换播放

SUPPORT - 辅助

REORDER-重排序

  • 将选中的图层反转图层顺序
  • Alt+REORDER，将选中的图层随机重排序

BAKER-烘培

  • 将表达式动画烘焙成关键帧

  • Alt+BAKER，删除表达式

TANGENTS-

RESIZER-重设尺寸

重新设置某个图层的尺寸

SLICER-切分

将图层按一定规则切分

选中图层，然后点击SLICER

SWAPPER

  • 将选中的所有图层替换为最后加入合成的图层，并保持当前大小不变
  • Alt+，非合成转换为预合成
  • Ctrl+，导入一个新资源到当前合成
