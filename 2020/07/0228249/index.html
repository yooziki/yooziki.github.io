    1 可以使用的工具

    有道云笔记

    虽然看起来这个软件不那么像编程软件 XD
    但它可以实时显示你MarkDown编写出来的内容
    还有一些简单的导航功能可以使用，对于初学MarkDown的人来说是非常的友好
    不过想在这个软件里插入图片预览，那是需要开通会员的

    VS + Markdown Preview Enhanced

    Markdown Preview Enhanced下载

    功能和有道云笔记差不多，甚至VS启动速度比有道云慢一些，没有了提示导航栏，不过图片插入不需要花钱开会员

    nodepad++

    写短代码最实用的工具
    除了不能预览

    Typora

    Typorax下载

    功能最强大的markDown编辑器，能够直接可视化的编辑

    对齐

    在表头下方的分隔线标记中加入:,即可标记下方单元格内容的对其方式：

    • :---代表左对齐
    • :--:代表居中对其
    • ---:代表右对齐
    left center right
    aaaa bbbbbb ccccc
    a b c

    不使用对其标记时，单元格中的内容默认左对齐；表头单元格中的内容会一直居中对齐

    插入其他内容

    表格中可以插入其它MarkDown中的行内容标记
