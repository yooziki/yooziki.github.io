知识点

turtle模块:

  • python绘图库turtle

  • 屏幕坐标系，窗体坐标系

  • turtle.setup(width,height,startx,starty):

    • 初始化启动窗体的位置和大小
    • 初始化不是必须函数
    • 后两个参数可以选择性填写，不填写会生成在屏幕正中心

  • turtle空间位置坐标体系

  • turtle.goto(x,y):

    • 从海龟当前位置到达指定坐标点(绝对坐标)

  • turtle.fd(d):

    • 海龟向前划出d距离

  • turtle.bk(d):

    • 海龟向后划出d距离

  • turtle.circle(r,angle):

    • 以海龟左侧r距离的点为原型，划出angle角度的弧线

  • turtle角度坐标体系

  • turtle.seth(angle)：

    • 改变海龟行进方向，angle为绝对角度
    • 不产生任何绘制

  • turtle.leff(angle)&turtle.right(angle):

    • 改变海龟行进方向，angle为海龟角度，向左或者向右转angle角度
    • 不产生任何绘制

  • RGB色彩体系:

    • R(red)G(green)B(blue)三种颜色通道组成，构成视觉颜色的万物色

    • 是计算机中最常见的色彩体系

    • 英文名称 RBG整数值 RGB小数值 中文名称
      white 255,255,255 1,1,1 白色
      yellow 255,255,0 1,1,0 黄色
      magenta 255,0,255 1,0,1 洋红
      cyan 0,255,255 0,1,1 青色
      blue 0,0,255 0,0,1 蓝色
      black 0,0,0 0,0,0 黑色
      seashell 255,245,238 1,0.96,0.93 海贝色
      gold 255,215,0 1,0.84,0 金色
      pink 255,192,203 1,0.75,0.80 粉红色
      brown 165,42,42 0.65,0.16,0.16 棕色
      purple 160,32,240 0.63,0.13,0.94 紫色
      tomato 255,99,71 1,0.39,0.29 番茄色

  • turtule.colormode(mode):

    • 改变海龟色彩模式（默认模式为小数值）
    • mode为1.0：RGB小数值模式
    • mode为255：RGB整数值模式

  • 画笔控制函数

    • turtle.penup()
      • turtle.pu()
      • 画笔抬起，抬起后海龟的运行轨迹不进行绘制
    • turtle.pendown()
      • turtle.pd()
      • 画笔放下，放下后海龟的运行轨迹将进行绘制
    • turle.pensize(width)
      • turtle.width(width)
      • 设置海龟的宽度（画笔大小）
    • turtle.pencolor(color)
      • color可以是颜色字符串，如turtle.pencolor("purple")
      • color可以是RGB的小数值(或整数值)，如turtle.pencolor(0.1,0.22,1)
      • color可以说RGB的元祖值，如turtle.pencolor((0.1,0.22,1))

  • 画笔运动控制函数

    • 控制行进方向
    • turtle.fd(d)
    • turtle.circle(r,angle)

  • 画笔方向控制函数

    • turtle.setheading(angle)
      • turtle.seth(angle)
      • 改变行进方向，海龟走角度
      • 绝对角度
    • turtle.left(angle)&turtle.right(angle)
      • 海龟自身角度

  • turtle.done():

    • 保证程序绘制完成后不会自动退出

循环语句：

  • 按一定次数循环执行的一组语句

  • for <变量> in range(<参数>):
	<被循环执行的一组语句>

知识点

整数类型

  • 十进制：1010，99，-2174
  • 二进制，以0B或者0b开头：0b0110，-0B0101
  • 八进制，以0o或者0O开头：0o123，-0O456
  • 十六进制，以0x或者0X开头：0x9a，-0X89c

浮点数

  • 带有小数点及小数的数字

  • 浮点数的取值范围和小数精度都存在限制，但常规计算可以忽略

  • 取值范围数量级约为-10308至10308，精度数量级10-16

  • 浮点数将运算存在不确定尾数

    • 0.1(十进制表示)
    • 0.00011001100110011001100110011001100110011001100110011001100110011010（二进制表示）
    • 0.1000000000000000055511151231257827021181583404541015625（转换成十进制表示）
    • 二进制表示小数，可以无限接近但不完全相同
    • 所以小数运算可能存在尾数

  • 浮点数比较的处理

    • >>> 0.1 + 0.2 == 0.3
False
>>> round(0.1+0.2, 1) == 0.3
True

字符串类型及操作

  • 两类四种表示方式

    • 一对单或双表示单行字符串
    • 一对三单或三双引号表示多行字符串

  • 切片高级用法

    • 可缺失<字符串>[M：N]，M缺失表示从开头，N缺失表示到结尾
      • "abcdef"[:3]表示“abc”
    • <字符串>[M:N:K]，根据步长K对字符串切片
      • "abcdefghij"[1:7:2]表示"bdf"
      • "abcdefghij"[::-1]表示"jihgfedcba"

  • 转意符\

    • 转意符表达特定字符的本意
      • "这里有个双引号(\")"结果为 这里有个双引号(")
      • 转意符能形成一些组合，表达一些不可打印的含义
      • \b回退，\n换行（下行行首），\r回车（本行首）
      • \t制表符tab

random模块

  • random库是使用随机数的Python标准库

  • 伪随机数：采用梅森旋转算法生成的(伪)随机序列中元素

  • 函数 描述
    seed(a=None) 初始化给定的随机种子，默认为当前时间
    >>>random.seed(10) #产生种子10对应的序列
    种子确定的时候，序列是一定的，不给定种子的时候，
    会调用系统当前时间（几乎无法复现）
    random() 生成一个[0.0,1.0)之间的随机小数
    >>>random.random()
    0.5714025946899135
    种子确定的时候，random会从种子产生的随机数序列中，按使用次数的顺序取值
    扩展随机数函数
    randint(a,b) 生成一个[a,b)之间的整数
    >>>random.randiint(10,100)
    64
    randrange(m,n[,k]) 生成一个[m,n)之间以k为补偿的随机整数
    >>>random.randrange(10,100,10)
    80
    getrandbit(k) GetRandBit，生成一个k比特长的随机整数
    >>>random.getrandbit(16)
    37885
    uniform(a,b) 生成一个[a,b)之间的随机小数
    >>>random.uniform(10,100)
    13.096321648808136
    choice(seq) 冲序列seq中随机选择一个元素
    >>>random.choice([1,2,3,4,5,6,7,8,9])
    8
    shuffle(seq) 将序列seq中元素随机排列，返回打乱后的序列
    >>>s=[1,2,3,4,5,6,7,8,9]
    >>>random.shuffle(s)
    >>>print(s)
    [3,5,8,9,6,1,2,7,4]

知识点

###　函数

  • 函数是一段具有特定功能的、可重用的语句组

  • 函数是一种功能的抽象，一般函数表达特定功能

  • 降低编程难度，代码复用

    • 函数定义时，所指定的参数是一种占位符
    • 函数定义后，如果不经过调用，不会被执行
    • 函数定义时，参数是输入、函数体是处理、结果是输出（IPO）

代码复用与模块化设计

  • 把代码当成资源进行抽象
    • 代码资源化：程序代码是一种用来表达计算的“资源”
    • 代码抽象化：使用函数等方法对代码赋予更高级别的定义
    • 代码复用：同一份代码在需要时可以被重复使用
  • 模块化设计
    • 通过函数或对象封装将程序划分为模块及模块间的表达
    • 具体包括：主程序、子程序和子程序间关系
    • 模块内部紧耦合，模块之间松耦合
      • 紧耦合：两个部分之间交流很多，无法独立存在
      • 松耦合：两个部分之间交流较少，可以独立存在

函数递归

  • 函数定义中调用函数自身的方式

  • 两个关键特征：

    • 链条：计算过程存在递归链条

    • 基例：存在一个或多个不需要再次递归的基例

    PyInstaller库

    • 第三方库

  • CMD下 pip install pyinstaller或者pycharm最下方找到Terminal，输入 pip install pyinstaller

  • 继续在CMD或者Terminal中输入
    pyinstaller -F <文件名.py>
    如果要更改程序图表
    pyinstaller -i <图标文件名.ico> -F <文件名.py>

    • 在侧边文件管理窗口中，dist下会生成一个同名的.exe可执行文件(Win系统中)，也可以在文件管理器中查看，双击可以打开程序，Ctrl+C终止

    • 参数 描述
      -h 查看帮助
      –clean 清理打包过程中的临时文件
      -D, --onedir 默认值，生成dist文件夹
      -F, --onefile 在dist文件夹中只生成独立的大包文件
      -i <图标文件名.ico> 指定打包程序使用的图标(icon)文件
