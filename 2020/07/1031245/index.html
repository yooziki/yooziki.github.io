知识点

组合数据类型

集合类型

  • 多个元素的无序组合

  • 每个元素是唯一的

  • 集合中的每个元素都是不可更改的

  • 用{}表示，元素之间用，，可以用set()建立集合

  • 集合运算“交并差补”

    操作符 描述
    S | T 返回一个新集合，包括在集合S和T中的元素 ，“并”
    S - T 返回一个新集合，包括在集合S但不在T中的元素 ，“差”
    S & T 返回一个新集合，包括同时在集合S和在集合T中的元素 ，“交”
    S ^ T 返回一个新集合，包括集合S和T中的非相同元素 ，“补”
    S<=T 或 S<T 返回True/False，判断S和T的子集关系
    S>=T 或 S>T 返回True/False，判断S和T的包含关系
    S |= T 更新集合S，将产生的并集给S
    S -= T 更新集合S，将产生的差集给S
    S &= T 更新集合S，将产生的交集给S
    S ^= T 更新集合S，将产生的补集给S

  • 集合处理方法

    • 操作函数或方法 描述
      S.add(x) 如果x不在集合S中，则向S中添加一个元素x
      S.discard(x) 如果x在集合S中，则移除元素x
      S.remove(x) 移除元素x，如果集合S中没有x，异常KeyError
      S.clear() 移除集合S中所有元素
      S.pop() 返回S的第一个元素，并且移除这个元素，空集异常KeyError
      S.copy() 返回一个集合S 的副本
      len(S) 返回集合S 的元素数
      x in S和x not in S 判断元素x是否存在与集合S中，返回True/False
      set(x) 将其它类型的变量x转变为集合类型

  • 数据去重

序列类型

  • 具有先后关系的一组元素，基类类型

    • 字符串类型
    • 元组类型
    • 列表类型

  • 一维元素向量，元素类型可以不同

  • 元素由序号引导，通过下标访问序列的特定元素

  • 序列类型通用操作符

    操作符及应用 描述
    x in s 如果x是序列s的元素，返回True，否则返回False
    x not in s 如果x不是序列s的元素，返回True，否则返回False
    s + t 连接两个序列s和t
    s*n 或 n*s 将序列s复制n次
    s[i] 索引，返回序列s中的第i个元素，i是序列的序号
    s[[M]:[N]:[K]] 切片方式
    len(s) 返回序列s的长度
    min(s) 返回序列s中最小元素，s中元素需要可比较
    max(s) 返回序列s中最大元素，s中元素需要可比较
    s.index(x)或
    s.index(x,i,j)    		 返回序列s冲i开始到j位置中，第一次出现x的位置
    s.count(x) 返回在序列s中出现元素x的总次数
    map(func, s) 将func函数，作用于s中的每个元素，无需for in循环
元组类型

  • 序列类型的拓展

  • 一旦创建不能修改

  • 使用小括号()或tuple()创建，元素间用逗号,分割

  • 可以使用或不使用小括号retrun 1,2返回的是（1,2）元组

列表类型
  • 序列类型的拓展
  • 创建后可以随意的修改
  • 使用方括号[]或者list()创建，元素间用逗号,分割
  • 列表中各元素类型可以不同，无长度限制
  • 函数或方法 描述
    ls[i] = x 替换列表ls第i元素为x
    ls[i:j:k] = lt 用列表lt替换ls切片后所对应元素子列表
    del ls[i] 删除列表ls中第i元素
    del ls[i:j:k] 删除列表ls中第i到第j以k为步长的元素
    ls += lt 更新列表ls，将列表lt元素增加到列表ls中
    ls *= n 更新列表ls，其元素重复n次
    ls.append(x) 在列表ls最后增加一个元素x
    ls.clear() 删除列表ls中所有元素
    ls.copy() 生成一个新列表，赋值ls中所有元素
    ls.insert(i,x) 在列表i处插入一个元素x
    ls.pop(i) 将ls中i位置的元素取出来，并且删掉i位置的元素
    ls.remove(x) 将ls列表中第一个出现的x元素删除
    ls.reverse() 将列表ls中的元素反转
    sorted(ls) 对ls进行临时排序

| ls.sort(reverse=False) | 排序并将结果覆盖ls |
| min(ls) | 找出最小值（纯数字列表） |
| max(ls) | 找出最大值（纯数字列表） |
| sum(ls) | 求和（纯数字列表） |

字典类型

  • 映射

    • 映射是一种（键）索引和（值）数据的对应
    • “addr” : “xxxx”

  • 字典是键值对的集合，键值对之间无序

  • 采用大括号{}和dict()创建，键值对用冒号:表示
    {<键1>:<值1>, <键2>:<值2>, ..., <键n>:<值n>, <键n+1>:<值n+1>}

  • 在字典变量中，通过键获得值
    <字典变量> = {<键1>:<值1>. <键2>:<值2>, ..., <键n>:<值n>}
    <值> = <字典变量>[<键>] <字典变量>[<键>] = <值>

函数或方法 描述
del d[k] 删除字典d中键k对应的数据值
k in d 判断键k是否在字典d中，如果是返回True，否则False
d.keys() 返回字典d中所有的键信息
d.values() 返回字典d中所有的值信息
d.items() 返回字典d中所有的键值对信息
d.get(k,<default>) 键k存在，则返回对应的值，否则返回<default>值
d.pop(k,<\default>) 键k存在，则取出对应的值，否则返回<default>值
d.popitem() 随机从字典d中取出一个键值对，以元组形式返回
d.clear() 清空字典中的所有键值对
len(d) 返回字典d中元素的个数

sorted(iterable, key=None, reverse=False)

iterable：可迭代对象，即组合数据类型

key：可以是函数或者lambda表达式，可迭代对象的每个元素都使用一次，获得的返回值进行排序

reverse：False从小到大, True从大到小

jieba库模块

  • 优秀的中文分词第三方库

  • cmd或者pycharm的terminal中pip install jieba

  • jieba分词三种模式

    • 精确模式：把文本精确的分开，不存在冗余
    • 全模式：把文本中所有可能的词语都扫描出来，有冗余
    • 搜索引擎模式：在精确模式的基础上，对长词再次切分
  • 函数 描述
    jieba.lcut(s) 精确模式，返回一个列表类型的分词结果
    >>>jieba.lcut(“中国是一个伟大的国家”)
    [‘中国’, ‘是’, ‘一个’, ‘伟大’, ‘的’, ‘国家’]
    jieba.lcut(s,cut_all=True) 全模式，返回一个列表类型的分词结果，但存在冗余
    >>>jieba.lcut(“中国是一个伟大的国家”, cut_all=True)
    [‘中国’, ‘国是’, ‘一个’, ‘伟大’, ‘的’, ‘国家’]
    jieba.lcut_for_search(s) 搜索引擎模式，返回一个列表类型的分词结果，存在冗余
    >>>jieba.lcut_for_search(“中华人民共和国是伟大的”)
    [‘中华’, ‘华人’, ‘人民’, ‘共和’, ‘共和国’, ‘中华人民共和国’, ‘是’, ‘伟大’, ‘的’]
    jieba.add_word(w) 向分词词库新增词w，只针对当前计算机当前安装的jieba库
    >>>jieba.add_word(“蟒蛇语言”)

知识点

文件

  • 文件是数据的的抽象和集合

    • 文件是存储在辅助存储器上的数据序列
    • 文件是数据存储的一种形式
    • 文件展现形态：文本文件和二进制文件
      • 文本文件和二进制文件只是文件的展示方式
      • 本质上，所有文件都是二进制形式存储
      • 形式上，所有文件采用两种方式展示

  • 文本文件

    • 由单一特定编码组成的文件，入UTF-8编码
    • 由于存在编码，也被看成是存储着的长字符串
    • 适用于例如：.txt文件、.py文件

  • 二进制文件：

    • 直接由比特0和1组成，没有统一字符编码
    • 一般存在二进制0和1的组织结构，即文件格式
    • 适用于例如：.png文件、.avi文件

  • 文件的打开
    <变量名（文件句柄）> = open(<文件名（路径和名称）>,<打开模式>)

    文件的打开模式 描述
    ‘r’ 只读模式，默认值，如果文件不存在，返回FileNotFoundError
    ‘w’ 覆盖写模式，文件不存在则创建，存在则完全覆盖
    ‘x’ 创建写模式，文件不存在则创建，存在则返回FileExistsError
    ‘a’ 追加写模式，文件不存在则创建，存在则在文件的最后追加内容
    ‘b’ 二进制文件模式
    ‘t’ 文本方式打开文件，默认
    ‘+’ 与r/w/x/a一同使用，在原功能基础上增加同时读写功能

  • 文件的关闭
    <变量名>.close()

  • 文件的读取

    操作方法 描述
    .read(size = -1) 读入全部内容，如果给出参数，读入前size长度
    >>>s = f.read(2)
    中国
    .readline(size = -1) 读入一行内容，如果给出参数，读入该行前size长度
    >>>s = f.readline()
    中国是一个伟大的国家！
    .readlines(hint = -1) 读入文件所有行，以每行为元素形成列表，
    如果给出参数，读入前hint行
    >>>s = f.readline()
    [‘中国是一个伟大的国家！’]

  • 文件的写入

    操作方法 描述
    .write(s) 向文件写入一个字符串或字节流
    >>>f.write(“中国是一个伟大的国家！”)
    .writelines(lines) 将一个元素全为字符串的列表写入字符
    >>>ls = [“中国”, “法国”, “美国”]
    >>>f.writelines(ls)
    中国法国美国
    .seek(offset) 改变当前文件操作指针的位置，offset翻译如下:
    0 - 文件开头；1 - 当前位置；2 - 文件结尾
    >>>f.seek(0) #回到文件头

  • 文件的添加与删除

    操作方法 描述
    os.remove() 删去某个文件，注意存在性检测
    os.rename(<原文件名>, <想要变更的目标名>) 两者都为路径，注意同名检测

  • 文件的移动和复制
    import shutil

    操作方法 描述
    shutil.copy(<原文件名>, <想要变更的目地名>) 两者都需要明确的文件路径，文件名可以重命名
    shutil.move(<原文件名>, <想要变更的目地名>) 和上者相同，不过这是移动

wordcloud模块库

  • 词云展示库

  • cmd或者pycharm的terminal中pip install wordcloud

  • 使用方法

    • 步骤1：配置对象参数
    • 加载词云文本
    • 输出词云文件

  • wordcloud.WordCloud(<参数>)

    参数 描述
    width 指定词云生成图片的宽度，默认400
    >>>w = wordcloud.WordCloud(width=600)
    height 指定词云生成图片的宽度，默认200
    >>>w = wordcloud.WordCloud(height=400)
    min_font_size 指定词云中的最小字号，默认4号
    >>>w = wordcloud.WordCloud(min_font_size = 10)
    max_font_size 指定词云中的最大字号，根据高度自动调节
    >>>w = wordcloud.WordCloud(max_font_size = 20)
    font_step 指定词云中字体字号的步进间隔，默认为1
    >>>w = wordcloud.WordCloud(font_step = 2)
    font_path 指定字体文件的路径，默认None
    >>>w = wordcloud.WordCloud(font_path = “msyh.ttc”) # 微软雅黑
    max_words 指定词云限时的最大单词数量，默认200
    w = wordcloud.WordCloud(max_words = 20)

知识点

自顶向下设计和自底向上执行

自顶向下

  • 解决复杂问题的有效方法
  • 将一个总问题表达为若干个小问题组成形式
  • 使用同样的方法进一步分解小问题

自底向上

  • 逐步组建复杂系统的有效测试方法
  • 分单元测试，逐步组装
  • 按照自顶向下相反的路径操作
  • 直至，系统各部分以组装的思路都结果测试和验证

程序设计思维

  • 抽象和自动化：
    • 计算思维：Computational Thinking
    • 抽象问题的计算过程，利用计算机自动化求解
    • 计算思维是基于计算机的抽象思维
    • 计算思维基于计算机强大的算力及海量数据
    • 抽象计算过程，关注设计和构造，而非因果
  • 编程是将计算思维编程现实的手段
  • Python计算生态的运用
    • 编程的起点不是算法而是系统
    • 编程如图搭积木，利用计算生态为主要模式
    • 编程的目标是快速解决问题

用户体验

  • 用户体验指用户对产品建立的主观感受和认识
  • 关心功能实现，更要关心用户体验，才能做出好产品
  • 编程只是手段，不是目的，程序最终为人类服务

程序设计模式

  • IPO

    • I:Input输入，程序的输入
    • P：Process处理，程序的主要逻辑
    • O：Output输出，程序的输出

  • 自顶向下设计

  • 模块化设计

    • 通过函数或对象封装将程序分为模块及模块间的表达
    • 具体包括：主程序、子程序和子程序之间的关系
    • 分而治之：一种分而治之、分层抽象、体系化的设计思想
    • 模块内部紧耦合，模块之间松耦合

  • 配置化设计：程序引擎+配置文件

    • 程序执行和配置分离，将可选参数配置化
    • 将程序开发编程配置文件编写，扩展功能而不修改程序
    • 关键在于接口设计，清晰明了、灵活可扩展

  • 应用开发的四个步骤：

    1. 产品定义：对应用需求充分理解和明确定义
      产品定义，而不仅是功能定义，要考虑商业模式

    2. 系统架构：以系统的方式思考产品的技术实现
      系统架构，关注数据流、模块化、体系架构

    3. 设计与实现：结合架构完成关键设计及系统实现
      结合可扩展性、灵活性等进行设计优化

    4. 用户体验：从用户角度思考应用效果
      用户至上，体验优先

路径操作

  • impor os.path as op

  • 函数 描述
    abspath(path) 返回path在当前系统中的绝对路径
    >>>os.path.abspath(“file.txt”)
    ‘C:\\Users\\…\\file.txt’
    normpath(path) 归一化path的表示形式，统一用\\分隔路径
    >>>os.path.normpath(“D://PYE//file.txt”)
    ‘D:\\PYE\\file.txt’
    realpath(path) 返回当前程序与文件之间的相对路径（relative path）
    >>>os.path.realpath(“C://PYE//file.txt”)
    ’…\\…\\…\\…\\…\\…\\file.txt’
    dirname(path) 返回path中的目录名称
    os.path.dirname(“D://PYE//file.txt”)
    ‘D://PYE’
    basename(path) 返回path中最后的文件名称
    os.path.basename(“D://PYE//file.txt”)
    ‘file.txt’
    join(path,*paths) 组合path与paths，返回一个路径字符串
    os.path.join(“D:/”,“PYE/file.txt”)
    ‘D:\\PYE\\file.txt’
    exists(path) 判断path对应文件或目录时候存在，返回True/False
    os.path.exists(“D:\\PYE\\file.txt”)
    False
    isfile(path) 判断path所对应时候为已存在的文件，返回True/False
    os.path.isFile(“D:\\PYE\\file.txt”)
    True
    isdir(path) 判断path所对应时候为已存在的目录，返回True/False
    os.path.isFile(“D:\\PYE\\file.txt”)
    False
    getatime(path) a->access,返回path对应文件按或目录上一次的访问时间
    os.path.getatime(“D:\PYE\file.txt”)
    1518356633.7551725
    getmtime(path) m->modify,返回path对应文件或目录最近一次的修改时间
    1518356633.7551725
    getctime(path) c->create,返回path对应文件或目录的创建时间
    os.path.getctime(“D:\PYE\file.txt”)
    ‘Sun Feb 11 21:43:53 2018’
    getsize(path) 返回path对应文件的大小，以字节为单位
    os.path.getsize(“D:\PYE\file.txt”)
