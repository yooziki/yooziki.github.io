一些常用正则

意义

能够表示一定意义的字符串的组合就是正则表达式。正则表达式就是一个字符串。

来源

  • 用来表示一定规则的表达式

  • unix操作系统——posix（bre和ere）

  • prel, pyton…其它语言都有独特的正则语法——pcre正则

prel——pcre； c——posix； php——pcre/posix； python——pcre

正则的作用

  • 字符串的过滤：在字符串获取需要的部分字符
  • 验证字符串：用户名、密码相关的输入框
  • 切割字符串：划分字符内容
  • 替换字符串：替换文章相关的信息
  • 爬虫中大量使用正则

正则的组成

原子，元字符，模式修正符

其它语言中可能存在定界符

原子

组成正则表达式的最小单位，一个正则表达式至少需要一个原子

  • 所有可见字符都是原子：a, b, c, d, e, … , A, B, C, D, …, 你, 我, …, +, -, !, %, …
  • 所有不可见字符也是原子：\t, \n,…\r
  • 正则专用转义字符：
    1. \d 表示0~9之间的任意一个字符 -> [0123456789]
    2. \D表示除了0~9之外的任意一个字符 ->[^0-9]
    3. \s 表示任意一个空白字符（不可见字符）->[ \n\r\t\v\f]
    4. \S表示任意一个非空白字符（可见字符）->[^ \n\r\t\v\f]
    5. \w表示0~9, a~z, A~Z和_中的任意一个字符(UNICODE下兼容万国语) ->[a-zA-Z0-9](不兼容万国语)
    6. \W 表示除了0~9, a~z, A~Z和_之外的任意一个字符 ->[^a-zA-Z0-9]
    7. . 表示任意一个字符[除了\n回车字符]

元字符[原子修饰符]

  • 在原子后面修饰原子的字符

  • []自定义原子列表

    • 表示自定义原子列表中的任意一个字符
    • 需要0~9中奇数组成的原子r'[13579]'
    • \d可以改写为[0123456789]
    • 如果多个字符在ASCII中连续，中间可以用-表示，例如\w改写为[a-zA-Z0-9]

  • [^]自定义排除列表

    • 表示原子列表之外的任意一个字符，剩下规则相同
    • /D可以改写为[^0-9]

  • 与数量相关的元字符

    元字符 描述
    + 被修饰的原子至少有一个，即数量为[1,∞]
    >>>print(\
    str(re.search(r’go+g’,“gg”))+\
    "\n"+str(re.search(r’go+g’,“gog”))+\
    "\n"+str(re.search(r’go+g’,“gooog”))\
    )
    None
    <re.Match object; span=(0, 3), match=‘gog’>
    <re.Match object; span=(0, 5), match=‘gooog’>
    ？ 被修饰的原子可有可无，即数量为[0,1];使用区间的时候问号表示非贪婪模式（匹配最少数量）
    >>>print(\
    str(re.search(r’go?g’,“gg”)) + \
    "\n"+str(re.search(r’go?g’,“gog”))+\
    "\n"+str(re.search(r’go?g’,“goog”))\
    )
    <re.Match object; span=(0, 2), match=‘gg’>
    <re.Match object; span=(0, 3), match=‘gog’>
    None
    * 被修饰的原子可以有任意多个，即数量为[0,∞]
    >>>print(\
    str(re.search(r’go*g’,“gg”))+\
    "\n"+str(re.search(r’go*g’,“gog”))+\
    "\n"+str(re.search(r’go*g’,“gooog”))\
    )
    <re.Match object; span=(0, 2), match=‘gg’>
    <re.Match object; span=(0, 3), match=‘gog’>
    <re.Match object; span=(0, 5), match=‘gooog’>
    {m} 被修饰的原子有m多个，即数量为m

    >>>print(\
    str(re.search(r’go{1}g’,“gg”))+\
    "\n"+str(re.search(r’go{1}g’,“gog”))+\
    "\n"+str(re.search(r’go{1}g’,“gooog”))\
    )
    None
    <re.Match object; span=(0, 3), match=‘gog’>
    None
    {m, n} 被修饰的原子有不少于m但不多于n个，即数量为[m,n]
    >>>print(\
    str(re.search(r’go{1,3}g’,“gg”))+\
    "\n"+str(re.search(r’go{1,3}g’,“gog”))+\
    "\n"+str(re.search(r’go{1,3}g’,“gooog”))\
    )
    None
    <re.Match object; span=(0, 3), match=‘gog’>
    <re.Match object; span=(0, 5), match=‘gooog’>
    {m, } 被修饰的原子不少于m个，即数量为[m,∞]
    {, n} 被修饰的原子不多于n个，即数量为[0,n]

  • 限定符号

    元字符 描述
    \A 限定内容必须在整个字符串的开头部位，放在所有原子之前，不支持多行匹配
    >>>print("{}\n{}".format(\
    str(re.search(r"\Agog",“gog”)),\
    str(re.search(r"\Agog",“ogog”))\
    ))
    <re.Match object; span=(0, 3), match=‘gog’>
    None
    \Z 限定内容必须在整个字符串的末尾部位，放在所有原子的最后，不支持多行匹配
    >>>print("{}\n{}".format\(
    str(re.search(r"og\Z",“gogog”)),\
    str(re.search(r"go\Z",“gogog”))\
    ))
    <re.Match object; span=(3, 5), match=‘og’>
    None
    ^ 限定内容必须在整个字符串的开头部位，放在所有原子之前，支持多行匹配模式
    >>>print("{}\n{}".format(\
    str(re.search(r"\Ao",“gog\no”,re.M)),\
    str(re.search(r"^o",“gog\no”,re.M))\
    ))
    None
    <re.Match object; span=(4, 5), match=‘o’>
    $ 限定内容必须在整个字符串的末尾部位，放在所有原子的最后，支持多行匹配模式
    >>>print("{}\n{}".format(\
    str(re.search(r"og\Z",“gog\no”,re.M)),\
    str(re.search(r"og$",“gog\no”,re.M))\
    ))
    None
    <re.Match object; span=(1, 3), match=‘og’>

  • 词边界

    元字符 描述
    \b 能够当成单词分割的字符，除了字母(汉字等)和数字都是词边界

    词边界可以加在原子的前和后
    >>>w = “Life is 12short12,I \nstudy python！”
    >>>print("{}\n{}\n{}\n{}\n{}".format(\
    re.search(r"Lif\b",w),\
    re.search(r"\bis",w),\
    re.search(r"\bshort\b",w),\
    re.search(r"\bstudy",w),\
    re.search(r"thon\b",w)\
    ))
    None
    <re.Match object; span=(5, 7), match=‘is’>
    None
    <re.Match object; span=(21, 26), match=‘study’>
    <re.Match object; span=(29, 33), match=‘thon’>
    \B 不能够当作单词分割的知乎(字母和数字都是非词边界)

    >>>w = “Life is 12short12,I \nstudy python！“
    print(”{}\n{}\n{}\n{}\n{}”.format(\
    re.search(r"Lif\B",w),\
    re.search(r"\Bis",w),\
    re.search(r"\Bshort\B",w),\
    re.search(r"\Bstudy",w),\
    re.search(r"thon\B",w)\
    ))
    <re.Match object; span=(0, 3), match=‘Lif’>
    None
    <re.Match object; span=(10, 15), match=‘short’>
    None
    None

  • 关系

    元字符 描述
    | 左右内容二选一，低优先级
    >>>print(re.search(r"ab|cd","abcd)
    <re.Match object; span=(0, 2), match=‘ab’>
    ( ) 改变匹配优先级
    >>>print("{}\n{}\n{}".format(\
    re.search(r"a(b|c)d",“abcd”),\
    re.search(r"a(b|c)d",“acd”),\
    re.search(r"a(b|c)d",“abd”)\
    ))
    None
    <re.Match object; span=(0, 3), match=‘acd’>
    <re.Match object; span=(0, 3), match=‘abd’>
    ( ) 将多个原子视为一个原子处理，方便元字符使用
    print("{}\n{}".format(\
    re.search(r"go+",“gogogogogogo”),\
    re.search(r"(go)+",“gogogogogogo”)\
    ))
    <re.Match object; span=(0, 2), match=‘go’>
    <re.Match object; span=(0, 12), match=‘gogogogogogo’>
    ( ) 将匹配的内容作为模式单元进行存储
    print("{}\n{}\n{}\n{}".format(\
    re.search(r"yooziki",“yooziki.site”),\
    re.search(r"yooziki",“yooziki.site”).groups(),\
    re.search(r"(yoozi)(ki)",“yooziki.site”),\
    re.search(r"(yoozi)(ki)",“yooziki.site”).groups()\
    ))
    <re.Match object; span=(0, 7), match=‘yooziki’>
    ( )
    <re.Match object; span=(0, 7), match=‘yooziki’>
    (‘yoozi’, ‘ki’)
    [ ] 括号内元素任选其一
    >>>print(re.search(r"[人生苦短]",“人生苦短”))
    <re.Match object; span=(0, 1), match=‘人’>

模式修正符

  • 设定匹配的一些额外的规则

    模式修正符 描述
    re.A(re.ASCII) 在ASCII模式下进行正则匹配操作
    re.U(re.UNICODE) 默认，在万国码UNICODE模式下进行正则匹配操作
    re.S(re.DOTALL) 取消原子.对\n的屏蔽
    re.M(re.MULTILINE) 多行模式，一般配合^和&
    re.X(re.VERBOSE) 匹配的时候忽略正则表达式中的空格或注释
    re.I(re.IGNORECASE) 忽略大小写匹配，UNICODE模式下，它将会匹配52个ASCII字符和4个额外的非ASCII字符： ‘İ’ (U+0130, 拉丁大写的 I 带个点在上面), ‘ı’ (U+0131, 拉丁小写没有点的 I ), ‘ſ’ (U+017F, 拉丁小写长 s) and ‘K’ (U+212A, 开尔文符号)
    re.L(re.LOCALE) 由当前语言区域决定\w,\W,\b,\B和大小写敏感度匹配。这个标记不推荐使用，因为语言区域机制很不可靠

  • 转义字符\的应用

    • 屏蔽原子、元字符和模式修正符的表达

  • 正则扩展语法

    扩展语法 描述
    (?Limsux) 多种模式修正符号的应用
    pattern = r"(?imxsu)yooziki"
    (?😃 取消单元存储功能，节省资源
    >>>print(re.search(r"yoozi(?:ki)", “yooziki.site”).groups())
    ( )
    (?P) 自定义模式单元的名称，以字典形式存储单元模式
    (?P=name) 根据name得到模式单元的对应的值
    (?#) 正则注释内容，解析器不会处理该内容
    (?=) 正向先行断言（非零宽断言）,从断言成立的右侧开始匹配正则表达式
    正向–>有；负向–>没有；先行–>向右(/前)进行查找；后行–>向左(/后)进行查找
    零宽–>宽度必须指定({m})，不能是可变宽度(\* ,+,{m,n},{m,},{,n})
    先行与后行可以同时存在
    (?!) 负向先行断言（零宽断言）
    (?<=) 正向后行断言（零宽断言）
    (?<!) 负向后行断言（零宽断言）
    (?(id/name)Y|N) 根据单元模式是否存在决定使用Y的规则(模式单元存在)或者是N的规则(模式单元不存在)

Re模块

re模块的方法

方法 描述
complie(<正则字符串>[,<模式修正符>]) 编译正则表达式获取正则表达式对象，显示转为对象，提高程序效率
>>>pattern = r’\d{12}’
>>>print(pattern,type(pattern),type(re.compile(pattern)))
\d{12} <class ‘str’> <class ‘re.Pattern’>
escape(<字符串>) 对字符串进行转义处理,对所有非数字和字母的字符转义处理
>>>s = “life is short, i study python!”
>>>print(re.escape(s))
life\ is\ short,\ i\ study\ python!
findall(<正则>, <匹配字符串>[, <模式修正符>]) 对字符串进行正则匹配，获取所有符合的内容结果，返回列表
>>>s = “life is short, i study python!”
>>>print(re.findall(r"i",s))
[‘i’, ‘i’, ‘i’]
finditer(<正则>, <匹配字符串>[, <模式修正符>]) 对字符串进行正则片匹配，获取所有符合的内容结果，返回迭代器地址，可以使用循环遍历每个对象
>>>s = “life is short, i study python!”
>>>print(re.finditer(r"i",s))
<callable_iterator object at 0x000002764C877A08>
>>>for i in re.finditer(r"i",s):
… print(i)
<re.Match object; span=(1, 2), match=‘i’>
<re.Match object; span=(5, 6), match=‘i’>
<re.Match object; span=(15, 16), match=‘i’>
match(<正则>,<匹配字符串>[,<模式修正符>]) 对字符串进行正则匹配，必须首字符匹配成功，获取第一个符合要求的内容结果，返回匹配结果对象
search(<正则>,<匹配字符串>[,<模式修正符>]) 对字符串进行正则匹配，获取第一个符合要求的内容结果返回匹配结果对象
split(<正则>,<切割字符串>[,<最大切割数量>]) 将正则匹配到的元素作为切分依据，切割字符串，返回列表
>>>s = “10十9九8八7七6六5五4四3三2二1一”
>>>print(re.split(r’\d+’,s,5))
[’’, ‘十’, ‘九’, ‘八’, ‘七’, ‘六5五4四3三2二1一’]
sub(<正则>,<要替换的字符串>,<被替换的字符串>[,<最大替换次数>]) 正则替换，将一个字符串替换进被正则匹配的字符串的匹配位置，返回字符串
>>>s = “人生苦短12341234123412341234”
>>>replace =“我学python”
>>>print(re.sub(r"(\d{4})",replace,s,3))
人生苦短我学python我学python我学python12341234
subn(<正则>,<要替换的字符串>,<被替换的字符串>[,<最大替换次数>]) 正则替换，将一个字符串替换进被正则匹配的字符串的匹配位置，返回元组(<替换字符串，替换次数>)
>>>print(re.subn(r"(\d{4})",replace,s))
(‘人生苦短我学python我学python我学python我学python我学python’, 5)

正则对象的方法

  • 正则对象通过re.compile(<正则对象>)获取

匹配操作

  1. re模块 (正则表达式，字符串)通过re模块方法得到结果对象
  2. 通过正则表达式对象，得到结果对象
方法 描述
O.findall(<要匹配的字符串>) 使用正则表达式对象O对指定的字符串进行匹配，获取所有匹配结果组成的列表
>>>s = “life is short, i study python!”

>>>O = re.compile(r"i")
>>>print(O.findall(s))
[‘i’, ‘i’, ‘i’]
O.finditer(<要匹配的字符串>) 对字符串进行正则片匹配，获取所有符合的内容结果，返回迭代器地址，可以使用循环遍历每个对象
O.match(<要匹配的字符串>) 对字符串进行正则匹配，必须首字符匹配成功，获取第一个符合要求的内容结果，返回匹配结果对象
O.search(<要匹配的字符串>) 对字符串进行正则匹配，获取第一个符合要求的内容结果返回匹配结果对象
O.split(<要切割的字符串>[,<最大切割次数>]) 将正则匹配到的元素作为切分依据，切割字符串，返回列表
O.sub(<要替换的字符串>,<被替换的字符串>[,<最大替换次数>]) 替换，将一个字符串替换进被正则匹配的字符串的匹配位置，返回字符串
O.subn(<要替换的字符串>,<被替换的字符串>[,<最大替换次数>]) 替换，将一个字符串替换进被正则匹配的字符串的匹配位置，返回元组(<替换字符串，替换次数>)

正则对象属性

属性 描述
O.flags 获取当前正则对象的模式修正符对应编号
>>>O = re.compile(r"(?imx)\d*")
>>>print(O.flags)
106
O.pattern 获取正则表达式的字符串格式
>>>O = re.compile(r"(?imx)\d*")
>>>print(O.pattern)
(?imx)\d*
O.groups 模式单元索引的个数
>>>O = re.compile(r"1(?P23)\d(\w)+")
>>>print(O.groups)
>>>O = re.compile(r"123\d\w+")
>>>print(O.groups)
2
0
O.groupindex 返回模式单元自定义的字典
>>>O = re.compile(r"1(?P23)\d(?P\w)+")
>>>print(O.groupindex)
>>>O = re.compile(r"1(23)\d(\w)+")
>>>print(O.groupindex)
{‘a’: 1, ‘b’: 2}
{}

正则对象扩展

  • 分片方式

  • 支持O.findall(s[, m, n])、O.finditer(s[, m, n])、O.math(s[, m, n])、O.search(s[, m, n])
