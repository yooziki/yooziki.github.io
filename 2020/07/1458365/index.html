Python系统操作OS模块

OS

operate System 操作系统import os加载模块

使用help(os)查看所有方法

方法

文件夹和文件操作

方法 描述
os.getcwd() get current work directory获取当前工作路径，即操作文件或者文件夹等的默认查找使用的路径，返回路径字符串
>>>print(os.getcwd())
G:\studyAbout\python\pyLevel1
os.chdir(path) change directory修改当前工作路径,无返回值

>>>os.chdir(‘D:’)
>>>print(os.getcwd())
D:\
os.listdir([path=DEFAULT]) 获取制定文件夹中的所有文件名和文件夹名组成的列表,返回列表，path可选，默认为当前工作路径
>>>print(os.listdir())
>>>print(os.listdir(os.getcwd()))
[’.idea’, ‘ATDconfig.txt’, ‘AutoTraceDraw.py’, ‘BatchInstall.py’,…]
os.mkdir(path[,<权限设置linux有效>]) make dirctory创建文件夹，无返回值
>>>os.mkdir("D:/新建文件夹)
os.makedirs(path[,<权限>]) 递归创建文件夹，自动创建路径中不存在的文件夹，无返回
os.,mkdir(“D:/a/b/c/d/e”) # 由于不存在"D:/a/b/c/d"所以e文件夹无法创建
os.makedirs(“D:/a/b/c/d/e”) # 自动创建路径中不存在的文件夹
Error
Done
os.rmdir(路径) remove directory删除空文件夹
>>>os.rmdir(“D:/a/b/c/d/e”)
os.removedirs(路径) 递归删除空文件夹
>>>os.removedirs(“D:/a/b/c/d/e”) # e->d->c->b->a依次删除到非空文件夹，文件夹不能处于占用
os.rename(,) 修改文件或文件夹名称,无返回值
>>>os.rename(“D:/te.txt”, “D:/test.exe”)
os.stat(path) 获取文件或者文件夹的状态信息,返回容器数据保存所有文件信息
path具体应用下下方详解
os.system(<系统指令>) 执行操作系统命令，不同os平台命令不同
谨慎操作
>>>os.system(“ping yooziki.site”)
[返回与命令行返回相同]

Linux权限

进制 所有者 所属组 其他人 权限表示 描述
八进制0o 文件夹或文件归谁所有，admin用户 所有者所在的组别，group用户 7 0o777 可读4可写2可执行1
可读可写4+2=6
可读可执行4+1=5
可写可执行2+1=3…

环境变量

  • 是一组路径的组合
  • 执行系统命令时，提供查找文件的路径
方法 描述
os.getenv(“path”) 获取系统的环境变量，返回环境变量信息字符串(以;分割)
os.getenv(“path”)
os.putenv(<需要的模块>，<新环境变量>) 向Python启动复制的系统环境变量中添加一段环境变量，无法用getenv取得
os.environ[<需要的模块>] 获取专用于python复制的环境变量，返回字典类型，使用[“path”]提取其中的path部分（大部分情况）
os.environ[<需要覆盖的模块>]=<新设置的环境变量> 专用于python的环境变量，使用=进行覆写

常用值

值 描述
os.curdir 获取当前文件夹的相对路径
.表示当前文件夹
os.pardir parent ，获取上层文件夹的相对路径
..表示上层文件夹,../../表示向上两层文件夹
os.path 获取当前文件的绝对路径，返回模块值
os.name 获取系统的标识符号
>>>print(os.name)
nt  # ----->windows操作系统
posix  #  ----->linux或者unix系统
os.sep 获取当前系统的路径分隔符号
>>>print(os.sep)
\ # ---->windows系统的路径分隔符
/ # ----->linux或者unix系统的路径分割符
os.extsep 获取文件名称与文件后缀名称之间的分隔符
所有操作系统都是点.
os.linesep 获取系统的换行符
>>>print(repr(os.linesep))
’\r\n’ # ----->windowns操作系统
’\n’  # ----->linux或者unix操作系统

路径详解

  • 相对路径：使用.或者..开头的路径:
    ./image/a.png(等价：image/a.png)
    …/abc/xyz
  • 绝对路径：完整描述路径的表示方式:
    C:/window/system32/calc.exe # windows
    /home/xdl/image # linux/unix

path子模块

  • 主要用于路径相关的操作
方法 描述
os.path.abspath(<相对路径>) 将相对路径转为绝对路径，返回绝对路径
>>>path = “a.png”
>>>print(os.path.abspath(path))
C:/Users/xxxx/a.png
os.path.basename(<路径>) 获取路径的主体部分
>>>path1 = “C:/Users/xxxx/a.png”
>>>print(os.path.basename(path1))
a.png
os.path.dirname(<路径>) 获取路径的路径部分
>>>path1 = “C:/Users/xxxx”
>>>print(os.path.dirname(path1))
C:/Users
os.path.join(<绝对路径部分1>，<相对部分2>) 将两个路径合并到一起
>>>path1 = “C:/window/abc”
>>>path2 = “xyz/123.txt”
>>>print(os.path.join(path1, path2))
C:/window/abc\xyz/123.txt
os.path.split(<路径>) 直接将路径拆分成路径部分和主体部分的元组
>>>print(os.path.split(“C:/window/abc/xyz/123.txt”))
(‘C:/window/abc/xyz’, ‘123.txt’)
os.path.splitext(<路径>) 讲路径拆分为文件后缀和其他部分组成的元组
>>>print(os.path.splitext(“C:/window/abc/xyz/123.txt”))
(‘C:/window/abc/xyz/123’, ‘.txt’)
os.path.getsize(<文件路径>) 获取文件的大小，返回字节数
>>>print(os.path.getsize(“C:/window/abc/xyz/123.txt”))
27648
os.path.isdir(<文件路径>) 判断是否是文件夹，如果是返回True否则False
os.path.isfile(<文件路径>) 判断是否是文件，如果是返回True否则False
os.path.islink(<路径>) 判断是否是链接，windows无效
os.path.getctime(<文件路径>) create获取文件创建时间，返回时间戳
os.path.getmtime(<文件路径>) modify获取文件最近修改时间，返回时间戳
os.path.getatime(<文件路径>) access获取文件最近访问数据，返回时间戳
os.path.exsits(<文件路径>) 检测文件路径是否真正存在，如果存在返回True，否则False
os.path.isabs(<文件路径>) 检测文件路径是否是一个绝对路径，如果是则返回True，否则False
os.path.samefile(<文件路径>) 检测两个路径是否指向同一个文件，如果是返回True，否则False
