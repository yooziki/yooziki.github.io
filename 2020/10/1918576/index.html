collections模块

发表于 2020-10-19 更新于 2020-11-17 分类于 语法 阅读次数： Valine：
本文字数： 3.6k 阅读时长 ≈ 3 分钟

官方文档

实现了特定目标的容器，以提供Python标准内建容器 dict,list,set,和tuple的替代选择

namedtuple

创建一个自定义的tuple对象，并且规定了tuple元素的个数，并可以用属性而不是索引来引用tuple的某个元素。

defaultdict

使用dict时，如果引用的key不存在，则抛出KeyError异常。

defaultdict会返回一个默认值

Counter

一个简单的计数器

例如，统计知乎出现的个数

deque

对list方法做了升级

更加适用于队列和栈

可以左插入和右插入，左弹出和右弹出 appendleft和popleft

功能 描述
append(x) 在列表右侧尾添加一个新元素x
appendleft(x) 在列表左侧头添加一个新元素x
pop() 从列表的右侧尾弹出一个元素
popleft() 从列表的左侧头弹出一个元素
clear() 清空列表
copy() 创建一份浅拷贝
count(x) 统计x元素在列表中出现的次数
extend(iterable) 右侧添加一系列元素
extendleft(iterable) 左侧添加一系列元素，iterable将会反过来
index(x[start[,end]]) 返回x在deque中的位置(可以设置起始和终止索引)
返回-第一个匹配项，如果未找到会引发ValueError
insert(i,x) 在i处插入x
如果该操作导致限长deque超出长度，会引发IndexError
remove(value) 移除找到的第一个value，如果一个都没有会引发ValueError
reverse 将deque反向
rotate(n) 向右循环移动n步
maxlen deque的最大尺寸，没有设置的话就是Node

OrderedDict

有序字典，其余和字典用法一致

可以比dict更擅长频繁的重新排序

由于内置的 dict类获得了记住插入顺序的能力（在 Python 3.7 中保证了这种新行为），它们变得不那么重要了。

功能 说明
popitem(last=True) 有序字典移除并返回一个键值对(key,value)
last=True-按照LIFO后进先出原则返回键值对
否则就按FIFO先进先出方式
move_to_end(key,last=True) 将现有key移动到有序字典的任意一端
last=True-移动到末尾，否则移动到开头
key不存在会触发KeyError

Iterable

可迭代对象iterable，字符串、列表、元组、字典、集合

Iterator

迭代器，用来表示一连串数据流的对象。迭代器从集合的第一个元素开始访问，截止到最后一个元素，只能前进不能后退

ChainMap

ChainMap 非常强大的dict字典组合功能，他将多个dict字典放入到一个list中，他比dict字典使用update快很多。通过ChainMap可以来模拟嵌套的情景，而且多用于模板之中。
